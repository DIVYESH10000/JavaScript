      43. What is JSON?

      

              
                1. JavaScript Object Notation: Not the same as JS object, but similar.

                2. Common in network calls and data storage.

                3. JSON.stringify() and JSON.parse()

                4. IMPORTANT :Strings are easy to transport over network.

                5. JSON requires double quotes. Escaped as \".
                
                6. IMPORTANT : JSON is data format, JS object is a data structure.
            
            Note: Because JSON is native to JS, we can say that JS object is by default a JSON Object. That's why we're able to directly pass JS Object to JSON.parse() and JSON.stringify().

                -In other languages(c object, python object,  java object etc) this does not happen, there we gotta manually convert them into JSON Object, THEN TO String.
                
            Example:

            // JS Object and JSON Object
            // Converting to String and back to Object using JSON.parse()  and JSON.stringify()

            console.log('\n\n43. What is JSON?\n\n');
            // JS Object
            let product = {
              name : 'T-Shirt',
              company : 'Mango',
              price : 666,

              rating : {
                stars: 4.5,
                noOfreviews: 500
              },
            };
            console.log(`typeof product before using JSON.stringify(product) : ${typeof product}\n\n`);
            console.log(product);
            // Converting JS Object into JSON String object using JSON.stringify(product);
            // Becomes JSON Object
            let str = JSON.stringify(product);
            console.log(`\n\ntypeof product after using JSON.stringify(product) : ${typeof str}\n\n`);
            console.log(str);

            // converting JSON Object back to JS Object using JSON.parse()

            let newProduct = JSON.parse(str);

            console.log(`\n\nconverting JSON Object back to JS Object using JSON.parse(str):${typeof newProduct}\n\n`);
            console.log(newProduct);

      45. Date

          

          1. new Date() :  Creates a new Date object with the current date
          and time.
          2. Key Methods:

          • getTime(): Milliseconds since Epoch.
          • getFullYear(): 4-digit year 
          • getDay(): Day of the week
          • getMinutes(): Current minute
          • getHours(): Current hour.

          3. Crucial for timestamps, scheduling, etc.
